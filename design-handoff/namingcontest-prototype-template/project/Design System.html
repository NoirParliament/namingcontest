Homepage Design System · v1.0

Your space for notes,
tasks, and big ideas.

A tokenised, screenshot-derived system for building a warm, paper-ish productivity homepage. Pastel feature panels on a beige page, Fraunces display + Inter body, pill-shaped controls, generous radii, and a hero framed by a soft sky.

01 · Foundations

A warm paper canvas, lifted by pastel cards.

Five principles drive every visual decision in the homepage. Read these first — every component below is just an expression of them.

1 · Beige base, never white

The page background is #fcf9f7 — a soft warm beige. White is reserved for cards, modals, and inset surfaces so they pop forward instead of disappearing.

2 · Big rounded panels

Major sections live inside 24–32px radius panels with a soft layered shadow. Think oversized index cards, not flat hero strips.

3 · Pastel categories

Each feature theme owns a tint: periwinkle = write, mint = imagine/connect, butter = plan. Tints repeat to teach the language; never invent a new one.

4 · Pill everything interactive

Nav, primary CTAs, secondary buttons, tags, even mini-cards inside panels — all use border-radius: 9999px for touch targets. Edges are soft; nothing is sharp.

5 · Serif headlines, sans body

Fraunces (display) carries every headline and every italic pull-quote. Inter handles body copy, eyebrows, UI labels. The contrast IS the brand.

6 · Paper collage atmosphere

The hero stages torn-paper shapes against a sky gradient. Decorative elements should feel cut and pasted — not vector-clean. Reserve for hero / between-section breaks.

05 · Radii & Shadows

Soft corners. Layered, almost-imperceptible shadows.

The system has only one sharp corner: the inside of a torn-paper element. Everything else rounds. The signature shadow-block is built from four stacked shadows so panels lift off the page without showing a visible drop.

08 · Token reference

Copy-paste CSS variables.

Drop these into :root and the rest of the system follows.

:root {
  /* surfaces */
  --bg-page:#fcf9f7; --bg-card:#ffffff; --bg-inset:#f4f1ef;
  --bg-ink:#030302;  --bg-overlay:#0303020a;

  /* ink */
  --fg:#030302; --fg-muted:#03030280; --fg-faint:#03030299;
  --fg-on-ink:#fff; --fg-on-ink-muted:#ffffffb3;

  /* lines */
  --border:#03030217; --ring:#0303021f;

  /* pastel panels */
  --panel-periwinkle:#b3c4f0; --panel-mint:#a6dcb3;
  --panel-butter:#fceebc; --panel-sky:#c4dffb; --panel-blush:#fadecc;

  /* accent solids (-4 family) */
  --accent-red:#bb433a; --accent-orange:#b25620; --accent-yellow:#987e1b;
  --accent-green:#3f8850; --accent-blue:#427e8a;
  --accent-purple:#4b68c3; --accent-pink:#a549ab;

  /* atmosphere */
  --hero-sky-top:#93cfed; --hero-sky-bottom:#c4e5f6; --promo-banner:#80c793;

  /* radii */
  --r-xs:6px; --r-sm:10px; --r-md:12px; --r-lg:16px;
  --r-xl:24px; --r-2xl:32px; --r-pill:9999px;

  /* spacing (4px) */
  --s-1:4px; --s-2:8px; --s-3:12px; --s-4:16px; --s-5:20px;
  --s-6:24px; --s-8:32px; --s-10:40px; --s-12:48px; --s-16:64px;
  --s-20:80px; --s-30:120px;

  /* type */
  --font-display:"Fraunces",ui-serif,Georgia,serif;
  --font-text:"Inter",system-ui,-apple-system,sans-serif;
  --tr-tight:-.025em; --tr-wide:.025em; --tr-wider:.05em; --tr-widest:.10em;

  /* shadows */

    0 50px 40px 0 #00000003,
    0 50px 40px 0 #00000005,

    0 3px 10px 0 #00000014;
  --shadow-card: 0 1px 2px #0000000a, 0 8px 24px #0000000f;
